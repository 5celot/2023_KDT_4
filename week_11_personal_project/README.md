## 🖥️ 프로젝트 소개
- 생성형 이미지 ai를 활용하여 어떠한 서비스들을 만들어 낼 수 있는지 알아보고 데이터를 이용해 서비스를 구현하였습니다.
- 프로젝트를 진행하기 앞서 생성형 모델들에 대해서 공부를 해 각 모델의 장단점을 알아 보았습니다.
