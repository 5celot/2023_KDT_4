# 🚩 9주차 팀 프로젝트
# **리뷰 분석 서비스**
----------------------------------------------------------

## 🖥️ 프로젝트 소개
- 판매 제품의 리뷰를 분석해 **감성분석**과 **시각화**를 통하여 개선해야 할 점을 찾는 서비스
- 네이버스토리, 올리브영, 쿠팡, 홈플러스 제품 url을 넣으면 **실시간**으로 리뷰를 크롤링 해서 분석과 결과를 도출
- 감성분석은 더 정확한 결과를 위해 감성사전을 이용하지 않고 **openai**를 사용해 분석
